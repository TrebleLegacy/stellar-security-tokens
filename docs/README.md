# Radox Platform — Documentation

> This directory contains all technical and operational documentation for the Radox platform.

## Reading Guide

### If you're new to the project
Start here, in order:

1. **[TOKENIZATION.md](TOKENIZATION.md)** — What the platform does: real-world assets → digital tokens.
2. **[SYSTEM_FLOW.md](SYSTEM_FLOW.md)** — End-to-end technical flow with Mermaid diagrams (onboarding → settlement).
3. **[AUTHENTICATION.md](AUTHENTICATION.md)** — How users log in (Passkeys for investors, MFA for admins).
4. **[INVESTMENT_FLOW.md](INVESTMENT_FLOW.md)** — How investors buy tokens.
5. **[PAYMENTS.md](PAYMENTS.md)** — How dividends/interest are distributed.
